✅ 예약 패키지 실행

날씨, 시장 동향, RSS 요약 패키지 설치 → cron 스케줄 → telegram setup 후 cron 시각에 발송.
weekly delivery live verified (commit af0c360).

✅ 작동 verified
시나리오

패키지가 데이터를 가져오고 메시지를 만듭니다

1

스킬 레지스트리에서 3개 설치

날씨 브리핑, 매크로 경제 리포트, RSS 뉴스 요약 패키지를 레지스트리에서 설치합니다.

Terminal
$ kittypaw skill install weather-briefing
Installed package "날씨 브리핑" (weather-briefing) v1.0.0 from registry
$ kittypaw skill install macro-economy-report
Installed package "매크로 경제 리포트" (macro-economy-report) v1.0.0 from registry
$ kittypaw skill install rss-digest
Installed package "RSS 뉴스 요약" (rss-digest) v1.0.0 from registry
2

패키지 설정 입력

날씨 패키지에 위치 좌표 입력. 텔레그램 봇 토큰은 kittypaw setup 의 대화형 wizard 로 한 번에. API 키 없이 동작 (Open-Meteo 무료).

Terminal
$ kittypaw setup # LLM 키 + 텔레그램 봇 채널 등록 (대화형)
$ kittypaw skill config weather-briefing latitude 37.57
$ kittypaw skill config weather-briefing longitude 126.98
3

cron 시각에 발송

각 패키지의 cron 에 따라 서버가 자동 실행 → LLM 요약 → 등록된 telegram 채널로 발송. 출근길에 폰 켜면 도착해 있음.
